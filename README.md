# Relógio do Centro de Controle — Android

Aplicativo Android nativo para exibir até cinco horários simultâneos, inspirado no painel de relógios de um centro de controle.

## Recursos

- Até cinco relógios configuráveis.
- Siglas de até três caracteres.
- Fusos horários com horário de verão quando aplicável.
- Sincronização por NTP, com fallback por HTTPS.
- Tela horizontal e modo imersivo.
- Mantém a tela ligada durante o uso.
- Configuração salva no aparelho.
- Toque longo na tela para abrir as configurações.

## APK

O GitHub Actions compila o APK de depuração automaticamente e o publica como artefato do workflow.
